# СГЛОБИ ДУМАТА 🦉

Игра-учебно помагало за дете на 5–6 г., което се учи да чете на български.
Сглобяваш думи по картинка-загадка: картинката е размазана или скрита, а всяка вярна
плочка я разкрива по малко. **16 нива** — 1–12 по **букви**, 13–16 по **срички**
(МА-МА, ПЕ-ПЕ-РУ-ДА) — звезди, подсказки и конфети.

**Играй тук:** https://borislavlazarov.github.io/sglobi-dumata/

Работи на телефон, таблет и компютър — без инсталация. Един офлайн HTML файл.
